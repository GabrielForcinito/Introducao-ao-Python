# 🐍 Comandos Básicos em Python

## 📌 Sobre o projeto

Este repositório reúne exercícios desenvolvidos durante o curso de Análise e Desenvolvimento de Sistemas (ADS), com foco na compreensão dos fundamentos da linguagem Python.

Os exercícios são introdutórios, mas fundamentais para a construção de uma base sólida em programação.

## 🛠️ Tecnologias utilizadas

* Python

## 🎯 Funcionalidades

* Execução de comandos básicos
* Entrada e saída de dados
* Manipulação de variáveis
* Exercícios práticos de lógica

## 📝 Desafios propostos

1. Exibir uma mensagem fixa na tela
2. Solicitar a profissão do usuário e exibir o valor informado
3. Solicitar a idade do usuário e apresentar na tela
4. Solicitar o sobrenome e exibir junto com uma mensagem personalizada
5. Solicitar o esporte favorito e exibir a informação

## 💻 Como executar

1. Certifique-se de ter o Python instalado
2. Baixe a pasta anexada
3. Abra a pasta com Vscode e execute os códigos desejados

## 📚 Aprendizado

Durante esses exercícios foram praticados os seguintes conceitos:

* Sintaxe básica do Python
* Tipos de dados
* Entrada e saída (`input` / `print`)
* Manipulação de variáveis
* Lógica de programação inicial
